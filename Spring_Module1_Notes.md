## Module1

### What is spring framework?
    - Spring is an opensource, Lightweight Dependency Injection container and framework for building Java enterprise applications
        -Don't have to worry about finding/connecting to each other
    - it is lightweight, so it doesn't need a runtime
    - it's non invasive
    - it's low overhead


### Spring is more than just a container

### Spring is a DI container
    - Spring will do all the heavy lifting for infrastructure necessary to develop applications:
        - Don't repeat yourself (DRY)
        - Seperation of concerns
        - Convention over configuration
        - Testability
